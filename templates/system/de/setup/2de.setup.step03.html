3. Schritt: Einrichten des Mailservers

In diesem Schritt richten Sie Ihren Mailserver ein. Die TechBox nutzt dabei einen Mailserver für ausgehende Email (zum Beispiel der Server Ihres Providers oder eines Freemailers), um Emails zu verschicken. Die TechBox kann Emails von Ihrem Miniserver entgegen nehmen und über diesen Mailserver versenden. Damit können aktuelle Einschränkungen des Miniservers (z. B. die verschlüsselte Übertragung) umgangen werden. Des weiteren werden alle Status-Emails der TechBox über diesen Account versendet.

Die notwendigen Daten, die Sie hier eintragen müssen, erfahren Sie von Ihrem Internet-Provider. Sie benötigen die Daten für den Server für ausgehende Email (oft auch als SMTP-Server bezeichnet). Alternativ können Sie auch einen Emailaccount eines Freemailers, wie zum Beispiel Google Mail/GMail, verwenden.

  An diese Emailadresse werden alle Status-Emails versendet.
  Rechnername Ihres Mailservers für ausgehende Email (SMTP Server).
  Portnummer des SMTP-Servers, normalerweise 25 oder 587.
  Setzt Ihr Mailserver eine verschlüsselte Übertragung voraus?
  Setzt Ihr Mailserver eine Authentifizierung per Username/Passwort voraus?
  Username für den SMTP-Server
  Passwort für den SMTP-Server
